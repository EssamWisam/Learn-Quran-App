[[


]] 
الَّذِي خَلَقَ فَسَوَّى { ۲ }
[[


]] 
وَالَّذِي قَدَّرَ فَهَدَى { ۳ }
[[


]] 
وَالَّذِي أَخْرَجَ الْمَرْعَى { ٤ }
[[


]] 
فَجَعَلَهُ غُثَاء أَحْوَى { ٥ }
[[


]] 
سَنُقْرِؤُكَ فَلَا تَنسَى { ٦ }
[[


]] 
إِلَّا مَا شَاء اللَّهُ إِنَّهُ يَعْلَمُ الْجَهْرَ وَمَا يَخْفَى { ٧ }
[[


]] 
وَنُيَسِّرُكَ لِلْيُسْرَى { ۸ }
[[


]] 
فَذَكِّرْ إِن نَّفَعَتِ الذِّكْرَى { ۹ }
[[


]] 
سَيَذَّكَّرُ مَن يَخْشَى { ۱۰ }
[[


]] 
وَيَتَجَنَّبُهَا الْأَشْقَى { ۱۱ }
[[


]] 
الَّذِي يَصْلَى النَّارَ الْكُبْرَى { ۱۲ }
[[


]] 
ثُمَّ لَا يَمُوتُ فِيهَا وَلَا يَحْيَى { ۱۳ }
[[


]] 
قَدْ أَفْلَحَ مَن تَزَكَّى { ۱٤ }
[[


]] 
وَذَكَرَ اسْمَ رَبِّهِ فَصَلَّى { ۱٥ }
[[


]] 
بَلْ تُؤْثِرُونَ الْحَيَاةَ الدُّنْيَا { ۱٦ }
[[


]] 
وَالْآخِرَةُ خَيْرٌ وَأَبْقَى { ۱٧ }
[[


]]

]]
